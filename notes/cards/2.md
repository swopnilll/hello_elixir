# Elixir Learning Notes – Creating Your First Project with Mix

## 1. Goal of This Section

Start writing actual Elixir code by creating a **new Elixir project** using the **Mix command line tool**.

---

# 2. What is Mix?

**Mix** is a **command-line tool that comes with Elixir**.
It helps developers manage Elixir projects.

Think of **Mix like a project manager for Elixir**.

### Mix can do many things:

* Create new projects
* Compile code
* Run tests
* Generate documentation
* Manage dependencies (libraries)
* Run different development tasks

So when you install **Elixir**, you automatically get **Mix** as well.

---

# 3. Elixir Environment (Simple View)

After installing Elixir, your computer contains:

```
Your Computer
   └── Elixir
         ├── elixir (language command)
         └── mix (project management tool)
```

* `elixir` → runs Elixir code
* `mix` → helps create and manage Elixir projects

---

# 4. Creating a New Elixir Project

Use the **mix new** command.

### Command

```bash
mix new cards
```

### Explanation

```
mix        → run the Mix tool
new        → create a new project
cards      → name of the project
```

---

# 5. Why the Name "cards"?

The project we are building will simulate **operations on a deck of playing cards**, such as:

* creating a deck
* shuffling cards
* dealing a hand
* saving/loading cards

So the project name **cards** matches the project purpose.

---

# 6. What Happens After Running the Command

Mix automatically **generates a project structure**.

Example output:

```
* creating README.md
* creating mix.exs
* creating lib/
* creating test/
```

This means Mix created:

* configuration files
* source code folders
* test folders

---

# 7. Next Step After Project Creation

Move into the project directory:

```bash
cd cards
```

Then view the files:

```bash
ls
```

You will see the generated project structure.

---

# Key Idea

In Elixir development:

```
Elixir → Programming language
Mix    → Tool used to create and manage Elixir projects
```

Almost every Elixir project starts with:

```
mix new project_name
```
