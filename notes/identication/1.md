# Identicon Project Setup

## Creating the Project

```bash
mix new identicon
cd identicon
```

## What is an Identicon?

An Identicon is a randomly generated image used as a default profile picture (e.g., on GitHub when no profile image is uploaded). Each user gets a unique, deterministic image based on their username or some input string.

## Identicon Specifications

- **Grid**: 5x5 squares
- **Dimensions**: 250px x 250px total
- **Square size**: 50px x 50px each
- **Symmetry**: Mirrored about the center vertical axis

```
+---+---+---+---+---+
| 1 | 2 | 3 | 2 | 1 |  <- columns 1,2 mirror to 4,5
+---+---+---+---+---+
```
