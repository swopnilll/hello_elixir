# Elixir File System + Erlang + BEAM (Notes)

## 🧠 Goal of This Section

* Save a **deck of cards** to the file system
* Later **load it back**
* Enables persistence (close app → reopen → restore state)

---

## 💾 Why Save to File?

* Store current deck on disk
* Reuse later without recreating
* Simple persistence mechanism

---

## 🔹 Elixir File System Support

* Built into **standard library**
* No extra packages needed
* Easy to read/write files

---

## ⚠️ Important Catch

* Not everything is purely Elixir
* Some functionality uses **Erlang under the hood**

---

## 🔹 Elixir vs Erlang

### Erlang

* Older language (~30+ years)
* Built for:

  * Telecom systems
  * High concurrency
  * Fault tolerance
* Syntax = harder / less friendly

---

### Elixir

* Modern language
* Cleaner, readable syntax
* Built **on top of Erlang**

✅ Think:

> Elixir = nicer interface for Erlang

---

## 🔄 How Code Actually Runs

1. You write **Elixir code**
2. Elixir **transpiles → Erlang code**
3. Erlang runs on **BEAM VM**

---

## ⚙️ What is BEAM?

* Stands for:
  **Bogdan/Björn’s Erlang Abstract Machine**
* A **virtual machine**
* Executes Erlang (and Elixir) code

---

## 🧩 Analogy

* Elixir → like Java
* BEAM → like JVM

---

## 🔹 Why This Matters Here

* File system operations:

  * Partly handled by **Erlang**
* So sometimes:

  * We call **Erlang functions from Elixir**

---

## 💡 Key Insight

> Elixir doesn’t reimplement everything
> It reuses Erlang’s powerful ecosystem

---

## 🔸 Important Notes

* You will see:

  * Erlang-style modules occasionally
* Don’t panic:

  * Usage is minimal and simple
* You don’t need deep Erlang knowledge

---

## 🔜 Next Step

* Use:

  * Elixir + a bit of Erlang
* Implement:

  * Save deck to file
  * Load deck from file

---

## 🧠 Takeaways

* Elixir runs on Erlang
* Erlang runs on BEAM
* File system = partly Erlang-powered
* You’ll occasionally use Erlang functions
* Complexity is **very small in practice**

---

👉 Big picture:
You’re not just learning Elixir — you’re tapping into the entire Erlang ecosystem.
