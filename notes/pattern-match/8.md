## 🚀 Overview

This project is a simple **Elixir module** to work with a deck of cards.

You can:

* Create a deck
* Shuffle it
* Deal cards
* Save to file
* Load from file

---

## 📦 Complete Code (Clean Version)

> ✅ Only **one `create_deck`** (final version)

```elixir
defmodule HelloElixir.Cards do

  # Create a full deck
  def create_deck do
    values = [
      "Ace", "2", "3", "4", "5", "6", "7",
      "8", "9", "10", "Jack", "Queen", "King"
    ]

    suits = ["Spades", "Hearts", "Diamond", "Clubs"]

    for value <- values, suit <- suits do
      "#{value} #{suit}"
    end
  end

  # Check if deck contains a card
  def contains?(deck, card) do
    Enum.member?(deck, card)
  end

  # Deal cards (returns tuple)
  def deal(deck, number_of_cards) do
    Enum.split(deck, number_of_cards)
  end

  # Shuffle deck
  def shuffle(deck) do
    Enum.shuffle(deck)
  end

  # Save deck to file
  def save(deck, filename) do
    binary = :erlang.term_to_binary(deck)
    File.write(filename, binary)
  end

  # Load deck from file
  def load(filename) do
    case File.read(filename) do
      {:ok, binary} ->
        :erlang.binary_to_term(binary)

      {:error, _reason} ->
        "That file does not exist"
    end
  end

end
```

## ⚠️ Common Mistakes

### ❌ Nested list (wrong comprehension)

```elixir
for suit <- suits do
  for value <- values do
    "#{suit} - #{value}"
  end
end
```

👉 Produces **list of lists**

---

### ✅ Correct

```elixir
for value <- values, suit <- suits do
  "#{value} #{suit}"
end
```
